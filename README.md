# You Are That Girl ✨

A visually stunning, empowering website designed to hype up and motivate Gen Z girls. Step into your power with daily affirmations, track your glow-up progress, and set the vibe with curated music.

## Features

- **Daily Affirmations & Compliments**: Get your daily dose of motivation and confidence boost with just a click.
- **Glow-Up Tracker 🪴**: Build healthy habits with a daily tracker for hydration, movement, skincare, and mindfulness. Track your current streak!
- **Interactive UI**: Enjoy a modern, glassmorphism design with a satisfying dark/light mode toggle.
- **Music Vibes 🎧**: A built-in Spotify player with curated playlists for your main character moments, plus a Daily Song discovery feature.
- **Sound Effects & Particle Animations**: Delightful micro-interactions and audio feedback enhance the overall experience.

## Getting Started

1. Clone this repository to your local machine.
2. Open `index.html` in your web browser.
3. Start tracking your habits, reading affirmations, and feeling like *That Girl*!

## Technologies Used

- HTML5
- CSS3 (Vanilla, custom properties, glassmorphism, animations)
- JavaScript (Vanilla, DOM manipulation, localStorage for streaks/saves)
- FontAwesome Icons
- Google Fonts (Outfit)
